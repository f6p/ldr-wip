Ladder8 v2
==========

Next L8 release. Main features:

* Competitive and Casual game types
* Competitive, Casual and Guest (aka as Unregistered) player accounts
* Game confirmations abandoned if favour of revoke mechanism
* Glicko2 rating system (instead of TrueSkill)
* Improved game reporting interface
* Improved rating synchronization with old ladder
* Ladder standings can be scoped by any time period
* Optional game replay processing
* Support for games of type other than 1vs1 and 2vs2

Repository changed from ladder8 to ldr-wip.
